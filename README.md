# python_entitas
# 简单介绍
该工程是使用python来生成python_entitas代码的简单工程, 目前支持 Context, Entity 正常代码生成( single, index, primaryIndex的生成).
注意,  作者的使用的地方是纯工具开发, 所以 Entitas的源码实现是没有考虑性能的. 如果有需要考虑性能的同学使用, 可能需要对源码进行修改使用.
